
## Project Overview: Image Processing Service

A simple Node.js image processing service that fetches an image from a remote URL,
applies transformations, and returns the processed image.

## Local set up
- npm install
- npm run start
- Test here: http://localhost:8082/?image_url=https://picsum.photos/id/870/200/300?grayscale&blur=2
